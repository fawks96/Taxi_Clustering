# Taxi_Clustering
Clustering taxi trajectory points in Shenzhen by DBSCAN

&ensp;

- **Dataset**

  Shenzhen taxi pick-up/drop-off points
  
- **Algorithm**

  DBSCAN (Density-Based Spatial Clustering of Applications with Noise)
  
- **Result**

  Clusters of pick-up / drop-off points

- **Usage**

  run **run.py**
